Regex Tutorial

Introduction

Regular expressions (regex) are powerful tools for pattern matching and text manipulation. In this tutorial, you will learn the basics of regex and how to use it effectively in JavaScript.

Summary

The regex featured in this tutorial is a basic email validation regex. It can be used to check if an email address is valid or not.

Component 1: Start anchor (^)

This component indicates the start of the email address. It ensures that the matching pattern starts at the beginning of the string.

Component 2: Domain name ([a-zA-Z0-9.-]+)

This component matches the domain name part of the email address. It allows alphanumeric characters, hyphens, and dots.

Component 3: At symbol (@)

This component matches the at symbol in the email address, separating the local part and the domain name.

Component 4: Local part ([a-zA-Z0-9._%+-]+)

This component matches the local part of the email address. It allows alphanumeric characters, dots, underscores, percent signs, and plus/minus symbols.

Component 5: Dot symbol (.)

This component matches the dot symbol in the email address, separating the local part and the top-level domain.

Component 6: Top-level domain ([a-zA-Z]{2,})

This component matches the top-level domain part of the email address. It allows two or more alphabetical characters.

Component 7: End anchor ($)

This component indicates the end of the email address. It ensures that the matching pattern ends at the end of the string.
